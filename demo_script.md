# 演示视频脚本

**时长目标**：60 秒
**录制方式**：Win+G 或 OBS 录屏
**准备**：打开一个记事本作为粘贴目标，程序启动好再开始录

---

## 开场（0-5s）

**画面**：桌面，记事本打开，右侧蓝色浮动麦克风按钮
**旁白**：语音输入助手，一款轻量级的 Windows 语音输入工具

---

## 第一幕：基本使用（5-25s）

**操作**：
1. 点击蓝色按钮（变红 = 录音中）
2. 说：「今天天气真不错，我想去公园散步」
3. 再点击按钮（变黄 = 处理中 → 变回蓝色）
4. 文字自动出现在记事本里

**旁白**：点击按钮开始录音，再次点击停止，文字自动粘贴到光标位置，GLM 大模型自动纠错和添加标点

---

## 第二幕：流式实时预览（25-45s）

**操作**：
1. 点击按钮开始录音
2. 说一段稍长的话：「语音识别技术在日常生活中越来越重要，从智能音箱到手机输入法都在广泛使用」
3. 录音时观察按钮下方实时出现的文字
4. 点击停止 → 完整文字粘贴到记事本

**旁白**：支持流式识别，说话时实时显示部分识别文字，基于 FunASR 流式模型，六百毫秒粒度输出

---

## 第三幕：拖拽 + 多次使用（45-55s）

**操作**：
1. 拖动按钮换个位置
2. 再录一句：「比赛加油」
3. 展示多次使用后记事本里有多行文字

**旁白**：按钮可自由拖拽，位置自动记忆

---

## 结尾（55-60s）

**画面**：GitHub 仓库页面（PR 列表或 README）
**旁白**：项目开源，71 项测试全通过，28 个 PR 持续交付
