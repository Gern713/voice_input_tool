# 演示视频脚本

**时长目标**：60 秒
**录制方式**：Win+G 或 OBS 录屏
**准备**：打开一个记事本作为粘贴目标，程序启动好再开始录

---

## 开场（0-5s）

**画面**：桌面，记事本打开，右侧蓝色浮动麦克风按钮
**旁白**：语音输入助手，一款轻量级的 Windows 语音输入工具

---

## 第一幕：基本使用（5-20s）

**操作**：
1. 点击蓝色按钮（变红 = 录音中）
2. 说：「今天天气真不错，我想去公园散步」
3. 再点击按钮（变黄 = 处理中 → 变回蓝色）
4. 文字自动出现在记事本里，带标点

**旁白**：点击按钮开始录音，再次点击停止，文字自动粘贴到光标位置，GLM 大模型自动纠错和添加标点

---

## 第二幕：热词功能（20-40s）

**操作**：
1. 打开项目目录下的 hotwords.txt，展示里面的热词内容（Transformer、machine learning、deep learning、reinforcement learning）
2. 关闭文件，点击按钮录音
3. 说：「我系统学习了 machine learning、deep learning 和 reinforcement learning，并且深入理解了 Transformer 架构的原理」
4. 停止录音，展示识别结果中英文术语拼写正确

**旁白**：支持自定义热词，编辑 hotwords.txt 添加专业术语，ASR 和 GLM 双层增强，大幅提升专业词汇识别准确率

---

## 第三幕：拖拽 + 多次使用（40-50s）

**操作**：
1. 拖动按钮换个位置
2. 再录一句实用的话：「这个项目的源代码已经开源在 GitHub 上」
3. 展示记事本里有多行文字

**旁白**：按钮可自由拖拽，位置自动记忆，随时随处可用

---

## 结尾（50-60s）

**画面**：GitHub 仓库页面（PR 列表或 README）
**旁白**：项目开源，63 项测试全通过，基于 FunASR 本地识别加 GLM 大模型后处理，无需联网即可使用

---
